ExpressJS Movie API Overview

checkID Middleware

Checks if a movie with the given ID exists. If not, it returns a 404 error.

validateBody Middleware

Validates the request body to ensure it contains the required properties (Title and Year).

getAllMovies

Handles GET requests to retrieve all movies. Responds with a list of all movies in JSON format.

createNewMovie

Handles POST requests to create a new movie. Adds a new movie to the database and responds with the created movie.

findMovieByID

Handles GET requests to retrieve a movie by its ID. Responds with the movie data if found.

updateMovieByID

Handles PATCH requests to update a movie's details based on its ID. Responds with the updated movie data.

deleteMovieByID

Handles DELETE requests to remove a movie by its ID. Responds with a success message if the deletion is successful.
